March Madness Office Pools: Getting Started

How it works:

Users can create and join March Madness Pools with their friends and/or co-workers. Before the tournament starts, each member will fill out brackets where they will predict the winners of each round of the tournament.

Scoring Systems

There are a couple of scoring systems that you can choose from.

The most popular is the Default scoring system where each round doubles the points with each win: 1,2,4,8,16,32

There is also the Slow Build scoring system that is similar to the Default but the later rounds aren't weighed as heavilty: 1,2,3,4,6,10

For those that just want to see who can predict the most, Flat gives one point for each win no matter where it is: 1,1,1,1,1,1

Lastly, if none of these work for you, you can create your own with Custom!

How to Win:

Have the bracket with the most points at the end of the tournament

Throughout the tournament, game results will come in as each game is played. If you guessed that a team would win over another team you get a point. Usually you will gain more points the further you correctly guess a team would win(this depends on pool settings).

So using the default settings, if you correctly guess that Duke would win its first game, and they do, you get one point, if you correctly guess that Duke would win the championship, and they do, you get 32 points. The pool member whose bracket accumulated the most points by the end of the tournament wins!

Thats It!

Now go create a March Madness Pool and let us do all the hard work for you! Hurry up, the tournament starts on Mar 17th!
